# VibeTalk

**VibeTalk** — веб-приложение для общения в формате мессенджера, вдохновлённое Discord.

Платформа позволяет находить друзей, обмениваться сообщениями в реальном времени, отправлять изображения и файлы различных форматов, а также совершать личные голосовые и видеозвонки.

Кроме того, в VibeTalk предусмотрена система тематических серверов по интересам и задачам, где пользователи могут публиковать посты, объединяться в сообщества и проводить групповые звонки.
